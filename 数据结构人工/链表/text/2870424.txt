头插法建表   void CreatListF(LinkNode *&L,ElemType a[],int n) {     LinkNode *s;     L=(LinkNode *)malloc(sizeof(LinkNode));     L→next=NULL;  //创建头节点，其next域置为NULL     for(int i=0;i<n;i++)  //循环建立数据节点s     {         s=(LinkNode *)malloc(sizeof(LinkNode));         s→data=a[i];  //创建数据节点s         s→next=L→next;  //将s插入到原首节点之前、头节点之后         L→next=s;     } } 这个算法的时间复杂度为O（N）。 链表的节点顺序和逻辑顺序正好相反。
